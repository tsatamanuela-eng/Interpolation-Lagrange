# Interpolation-Lagrange
Programme d'interpolation de Lagrange avec deux options : saisir les nœuds et leurs valeurs ou  les nœuds et une fonction mathématique. Par suite, il génère un polynôme d'interpolation avec des coefficients présentés sous forme de  fractions, une visualisation claire de la courbe interpolée, les points d'origine et les estimations demandées.
